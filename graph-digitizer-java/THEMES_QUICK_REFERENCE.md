# Themes Menu Quick Reference

## What's New?

Your Graph Digitizer now has a **Themes menu** in the menu bar with 14 professional themes to choose from!

## How to Use

1. Click **Themes** in the menu bar
2. Select any theme from the list
3. Watch the entire application instantly change to the new theme!

## Available Themes

### Light Themes (Good for daytime use)

- **FlatLaf Light** - Clean, bright white background
- **FlatLaf IntelliJ** - Professional light gray background
- **Atom One Light** - Subtle light theme
- **Solarized Light** - Warm, easy-on-eyes light theme

### Dark Themes (Good for low-light environments)

- **FlatLaf Dark** - Modern dark with neutral colors
- **FlatLaf Darcula** - IntelliJ's famous Darcula dark theme
- **Atom One Dark** - Warm dark theme
- **Dracula** - Popular vibrant dark theme
- **Material Design Dark** - Google Material Design dark
- **Nord** - Arctic-inspired cool dark theme
- **One Dark Pro** - VS Code's One Dark Pro theme
- **Solarized Dark** - Warm, balanced dark theme

### Colorful Themes

- **Arc** - Modern flat design with blue accents
- **Arc Orange** - Warm variant with orange accents

## Why Use Themes?

✨ **Personalization** - Choose the look you prefer  
👀 **Comfort** - Dark themes reduce eye strain in low light  
📊 **Accessibility** - Some themes work better with certain color vision needs  
🎨 **Professional** - 14 carefully-designed professional themes  

## Features

- 🖱️ Click to change instantly
- ⌨️ Fully keyboard accessible
- 🔊 Screen reader announces theme changes
- 📈 Works perfectly with all application features
- ⚡ Zero performance impact

## Tips

💡 **Pro Tip**: Try **Dracula** for dark environments or **Solarized Light** for reading lots of text
